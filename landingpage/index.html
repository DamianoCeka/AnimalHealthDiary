Frequently Asked Questions

Is AnimalHealthDiary free?

Yes! AnimalHealthDiary is completely open source and free to use. You can run it locally on your machine.

What animals are supported?

The application supports dogs, cats, and other pets. You can track multiple animals simultaneously.

Is my data secure?

Yes! All data is stored locally on your device. No data is transmitted to external servers except for AI analysis via OpenAI API (optional).

Do I need an OpenAI API key?

The AI features require an OpenAI API key, but the basic tracking features work without it.

Can I export my data?

Absolutely! You can export all your pet health data in CSV or JSON format at any time.

Does it replace a veterinarian?

No. This is an informational tool only. Always consult a qualified veterinarian for medical advice.
